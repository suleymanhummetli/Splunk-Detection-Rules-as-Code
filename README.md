# Splunk-Detection-Rules-as-Code
Automated OWASP Top 10 detection rules for Splunk Enterprise using GitHub Actions and REST API.
This repository manages Splunk Detection Rules using a "Detection as Code" approach.

Features:

15 Professional SIEM Rules (OWASP Top 10 focused).

Automated synchronization via Splunk REST API.

CI/CD integration with GitHub Actions.
